3 PDF 불러오기

PDF 준비하기

스캔한 설문지 PDF 파일들을 설문지 스캔 폴더나 다른 폴더에 모아주세요.

PDF 준비 꿀팁:
모든 설문지는 같은 양식(같은 문항, 같은 위치)이어야 합니다.
스캔은 200 DPI 이상으로 해주세요 (선명할수록 인식 잘 됨).
PDF 페이지는 설문지 1장 단위로 끊어주세요.

프로그램에서 불러오는 순서

  1. 프로그램 왼쪽 위 파일 버튼을 클릭합니다.
  2. PDF 불러오기를 클릭합니다.
  3. 컴퓨터에서 PDF 파일을 선택하고 열기 버튼을 누릅니다.
    Ctrl 키를 누르면 여러 파일을 한 번에 선택할 수 있습니다.
  4. 팝업창에 설문지 1부당 페이지 수를 입력합니다.
    예: 1장짜리 = 1, 2장짜리 = 2, 앞뒷면 = 2
  5. 잠시 기다리면 PDF가 화면에 나타납니다!
