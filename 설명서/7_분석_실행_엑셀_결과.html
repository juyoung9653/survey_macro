7 분석 실행 & 엑셀 결과

모든 설정(그룹, 값 매핑)이 끝났으면 이제 실제 분석을 실행할 차례입니다!

분석 실행

  1. 오른쪽 위 분석 실행 버튼(초록색)을 클릭합니다.
  2. 프로그램이 불러온 모든 PDF 파일을 하나씩 분석합니다.
    PDF 개수에 따라 1~5분 정도 소요됩니다.
  3. 분석 진행 상황이 프로그레스 바로 표시됩니다.
  4. 분석이 끝나면 설문지 스캔 폴더 안에 설문결과.xlsx 파일이 생성됩니다!
오른쪽 상단의 초록색 분석 실행 버튼

엑셀 결과 파일 내용

설문결과.xlsx 파일을 열어보면 다음과 같은 시트들이 있습니다:

엑셀 결과 시트 예시 (파일별, 문항별로 정리됨)

엑셀 시트별 설명

  • 결과: 모든 응답을 파일별로 한 줄씩 나열한 원시 데이터
  • 전체 통계: 모든 응답을 합친 문항별 선택 갯수와 비율(%)
  • 파일별 시트: 각 파일 이름으로 된 시트 - 해당 파일의 응답 통계만 별도로 확인
통계가 자동 계산됩니다!
엑셀 파일에는 SUM, COUNTIF 같은 함수가 미리 입력되어 있어서,
데이터만 바뀌어도 통계가 자동으로 업데이트됩니다.

주의: 결과 시트의 데이터를 수정하면
전체 통계 시트와 파일별 시트의 통계도 함께 자동으로 반영됩니다.
결과 시트가 모든 시트의 기준이 됩니다.
