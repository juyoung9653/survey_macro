5 그룹 만들기 (문항 묶기)

자동 찾은 체크박스들을 같은 문항끼리 묶어서 그룹으로 만들어야 합니다.
예: "만족도" 문항의 4개 체크칸을 하나로 묶기

그룹 만드는 순서

  1. 묶고 싶은 체크박스들을 좌클릭(작은 영역 드래그 또는 클릭)해서 선택합니다.
    선택된 박스는 노란색 굵은선으로 변합니다.
  2. 여러 개를 선택할 때는 Shift 키를 누른 상태로 좌클릭(작은 영역)하거나 우클릭 드래그로 영역을 지정하면 추가 선택됩니다.
  3. 원하는 박스를 모두 선택했으면 선택 묶기 (그룹화) 버튼을 클릭합니다.
  4. 팝업창에 그룹 이름을 입력합니다.
    예: "만족도", "청결도", "재방문의사" 등
  5. 확인을 누르면 하나의 그룹(문항)으로 묶입니다!
    묶인 박스들은 초록색 실선으로 표시되고 그룹명이 보입니다.
선택 → 선택 묶기 (그룹화) → 그룹명 입력
이해가 안 된다면 이렇게 생각하세요:
설문지 1번 문항에 체크칸이 4개(매우만족/만족/보통/불만족) 있죠?
이 4개의 체크칸을 하나로 묶어서 "이건 1번 문항이야"라고 알려주는 겁니다.

의견 칸 지정하기

체크박스 대신 글로 쓰는 의견란이 있다면, 해당 그룹을 의견 칸으로 지정할 수 있습니다.

  • 의견 칸으로 지정 버튼을 누르고 목록에서 그룹을 선택하세요.
  • 엑셀 결과 시트에는 해당 문항의 유무(작성 있음/없음)가 "O" / "X"로 기록됩니다.
  • 실제 작성된 의견 텍스트는 의견.pdf 파일로 따로 저장됩니다.
