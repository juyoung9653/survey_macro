10 자주 묻는 질문 (FAQ)

프로그램을 더블클릭했는데 아무 반응이 없어요.
파일 위에서 마우스 우클릭 > 연결 프로그램 > Python을 선택해보세요.
Python이 설치되어 있지 않다면 python.org에서 Python을 설치해야 합니다.
관리자에게 문의해주세요.
체크박스 자동 찾기가 너무 많이 또는 너무 적게 찾았어요.
자동 탐지는 완벽하지 않습니다. 직접 추가/삭제하면 됩니다.
좌클릭 드래그로 체크박스를 추가하고, 우클릭 > 삭제로 불필요한 박스를 지우세요.
스캔 품질이 좋을수록(200 DPI 이상, 선명하게) 인식률이 높아집니다.
여러 PDF를 한 번에 불러올 수 있나요?
네! 파일 선택 창에서 Ctrl 키를 누른 상태로 파일을 클릭하면 여러 개를 선택할 수 있습니다.
모든 PDF가 동일한 양식이어야 정확하게 분석됩니다.
설문지가 기울어져서 스캔됐는데 괜찮나요?
프로그램이 자동으로 정렬(이미지 정합)해서 보정합니다.
다만 너무 심하게 기울어지면 인식이 어려울 수 있으니, 스캔 시 최대한 반듯하게 넣어주세요.
엑셀 파일이 생성되지 않아요.
분석 실행 버튼을 눌렀는지 확인하세요.
분석 중 오류가 발생했다면 검은 창(터미널)에 빨간색 에러 메시지가 표시됩니다.
에러 내용을 관리자에게 알려주세요.
실행 중 검은 창이 갑자기 사라졌어요.
프로그램이 비정상 종료된 것입니다. 다시 더블클릭해서 실행해보세요.
같은 문제가 반복된다면 PDF 파일에 문제가 있는지 확인하거나 관리자에게 문의하세요.
다른 컴퓨터에서도 사용할 수 있나요?
설문지 스캔 폴더째로 USB에 담아서 다른 컴퓨터에 복사하면 됩니다.
단, 그 컴퓨터에 Python이 설치되어 있어야 하고,
첫 실행 시 인터넷 연결이 필요합니다 (필요 패키지 자동 설치).
문제가 계속된다면:
검은 창(터미널)에 출력된 에러 메시지를 캡처해서 관리자에게 보내주세요.
가장 빠르게 문제를 해결할 수 있습니다.

⚡ 빠른 시작 요약

  1. 이거_더블클릭해서_실행.py 실행
  2. 파일 > PDF 불러오기 > 설문지 PDF 선택
  3. 자동 찾은 체크박스 확인 + 직접 추가/삭제
  4. 체크박스 우클릭 선택 > 선택 묶기 버튼으로 그룹화
  5. 값 매핑 버튼으로 보기 이름 입력
  6. 분석 실행 버튼 클릭
  7. 설문결과.xlsx 열어서 확인!
