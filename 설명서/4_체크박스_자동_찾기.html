4 체크박스 자동 찾기

PDF를 불러오면 프로그램이 자동으로 설문지 안의 체크박스(네모 칸)를 찾아 파란색 실선 박스로 표시합니다.

실제 화면에서는 파란색 실선 박스로 표시됩니다

자동 그룹화 (문항 묶기)

체크박스를 찾은 후, 프로그램이 자동으로 가까운 체크박스끼리 묶어 그룹(문항)으로 만들어줍니다.
예: 1번 문항의 체크칸 4개가 자동으로 하나의 그룹으로 묶입니다.

  • 자동 그룹화된 박스는 초록색 실선으로 표시됩니다.
  • 그룹명은 자동으로 "문항 1", "문항 2" 형식으로 생성됩니다.
  • 자동 그룹화가 잘못되었다면 직접 그룹을 수정하거나 다시 묶을 수 있습니다. (다음 페이지 참고)
자동 그룹화 기준: 체크박스 사이의 거리와 위치를 분석하여 같은 줄에 가까이 있는 박스끼리 묶습니다.
설문지가 깔끔할수록 정확도가 높아집니다.

자동 탐지 실패 시 직접 추가

프로그램이 일부 체크박스를 못 찾았다면 직접 추가할 수 있습니다:

  • 마우스 왼쪽 버튼을 누른 채로 드래그하면 파란색 박스가 그려집니다.
  • 박스가 체크박스를 덮도록 적당한 크기로 그려주세요.
  • 잘못 그린 박스는 우클릭 드래그로 영역을 선택한 후 선택 박스 삭제 버튼을 누르면 지울 수 있습니다.
좌클릭 드래그로 박스 추가 / 우클릭 드래그로 영역 선택 후 삭제

선택된 박스 색상

  • 파란색 실선: 자동 탐지되었거나 직접 추가한 미그룹 박스
  • 초록색 실선: 그룹(문항)으로 묶인 박스
  • 노란색 굵은선: 현재 선택된 박스 (삭제/그룹화 대상)
화면 확대/축소: Ctrl 키를 누른 채로 마우스 휠을 돌리면 화면을 확대/축소할 수 있습니다.
정밀하게 박스를 그리고 싶을 때 유용합니다.
